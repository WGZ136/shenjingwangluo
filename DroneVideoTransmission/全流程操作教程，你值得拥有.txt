第一部分：软件安装
步骤1.1：安装Arduino IDE

打开浏览器，输入网址：https://www.arduino.cc/en/software

点击下载：找到"Windows Win 10 and newer"，点击"Windows ZIP file"

解压文件：下载完成后，右键点击ZIP文件 → "解压到当前文件夹"

运行程序：进入解压后的文件夹，双击arduino-ide.exe

步骤1.2：安装ESP32开发板支持（必须完成）

打开首选项：Arduino IDE中点击"文件" → "首选项"

添加网址：在"附加开发板管理器网址"框中，点击右侧的图标

粘贴网址：输入网址：https://espressif.github.io/arduino-esp32/package_esp32_index.json

点击确定：先点小窗口的"确定"，再点大窗口的"确定"

打开开发板管理器：点击"工具" → "开发板" → "开发板管理器"

搜索ESP32：在搜索框中输入esp32

安装开发板：找到"esp32 by Espressif Systems"，点击右侧的"安装"

等待完成：安装过程需要10-20分钟，看到"INSTALLED"表示成功

步骤1.3：针对ESP32 Camera库找不到的问题：

方法A：使用正确的搜索词（先试这个）

点击"工具" → "管理库"

在搜索框中尝试以下关键词（一个一个试）：

esp32-camera（注意中间有横线）

esp32cam

camera

ov2640

esp32 camera

如果找到类似"esp32-camera"的库，点击"安装"

方法B：手动安装库（如果方法A失败）

下载库文件：

打开浏览器，访问：https://github.com/espressif/esp32-camera

点击绿色的"Code"按钮

选择"Download ZIP"

保存到桌面（文件名为esp32-camera-master.zip）

在Arduino IDE中添加库：

打开Arduino IDE

点击"项目" → "加载库" → "添加.ZIP库"

选择刚才下载的esp32-camera-master.zip

点击"打开"

验证库是否安装成功：

点击"文件" → "示例"

向下滚动，看看有没有"esp32-camera"这个分类

如果有，说明安装成功

方法C：使用备选库（如果以上都失败）

在库管理器中搜索TinyPICO

安装"TinyPICO by Seon Rozenblum"

这个库包含了摄像头驱动，可以替代ESP32 Camera库

步骤1.4：验证安装是否成功

新建一个测试程序：

点击"文件" → "新建"

输入以下代码：

#include "esp_camera.h"

void setup() {
  Serial.begin(115200);
  Serial.println("ESP32 Camera库测试成功！");
}

void loop() {
  // 什么都不做
}

选择开发板：

点击"工具" → "开发板"

选择"ESP32 Arduino" → "ESP32S3 Dev Module"

点击验证：点击左上角的✓按钮

如果没有报错，说明一切正常！

第四部分：无人机安装

步骤4.1：安装原则：

前视摄像头：安装在无人机前方，镜头朝前

下视摄像头：安装在无人机底部，镜头垂直朝下

保持平衡：两个摄像头尽量对称安装

避免遮挡：不要挡住GPS、天线、螺旋桨

步骤4.2：供电方案

需要材料：

3.7V锂电池 ×2（给摄像头供电）

降压模块 ×2（将无人机电池电压降到3.3V）

接线方法：

无人机电池（7.4V/11.1V）
       ↓
   降压模块（降到3.3V）
       ↓
  ESP32-S3-CAM（3.3V）

具体接线步骤：

降压模块接线：

IN+（输入正极） → 无人机电池正极（红线）

IN-（输入负极） → 无人机电池负极（黑线）

OUT+（输出正极） → ESP32-S3-CAM的3.3V引脚

OUT-（输出负极） → ESP32-S3-CAM的GND引脚

调整电压：

用螺丝刀旋转降压模块上的电位器

用万用表测量OUT+和OUT-之间的电压

调整到3.3V（如果没有万用表，跳过此步）

步骤4.3：固定摄像头

清洁表面：用酒精棉片清洁无人机安装位置

粘贴双面胶：在摄像头背面贴上3M双面胶

撕掉保护膜：撕掉双面胶的保护层

粘贴摄像头：

前视摄像头：粘贴在无人机前方，镜头朝前

下视摄像头：粘贴在无人机底部，镜头朝下

加固：用扎带进一步固定，防止飞行时脱落

步骤4.4：连接电源

连接前视摄像头：

将降压模块1的OUT+连接到前视摄像头的3.3V引脚

将降压模块1的OUT-连接到前视摄像头的GND引脚

连接下视摄像头：

将降压模块2的OUT+连接到下视摄像头的3.3V引脚

将降压模块2的OUT-连接到下视摄像头的GND引脚

检查连接：

确保所有接线牢固

确保没有短路风险

用胶带固定裸露的线头
